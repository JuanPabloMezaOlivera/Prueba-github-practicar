# Comandos-Aprendidos-git
git init para iniciliazar git en el repositorio local  
git global--name.user/email  para configuar el usuario   
git log para ver el historial de commits en   
git  push subir  cambios al remoto   
git pull  para  trae y fusiona los cambios al repositorio local  
git add . preparar un archibo para git  
git commit -m  hacer un comentario del commit
